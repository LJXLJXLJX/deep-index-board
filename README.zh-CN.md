# Deep Index Board

[![CI](https://github.com/LJXLJXLJX/deep-index-board/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/LJXLJXLJX/deep-index-board/actions/workflows/ci.yml)

[English](README.md) | 简体中文

面向“粘贴前总要改一下”的剪贴板历史工具。

Deep Index Board 是一个带可编辑预览区的桌面剪贴板历史应用。你可以找回旧的剪贴板内容，在粘贴前临时修改文本，然后将修改版另存为新条目，或覆盖保存到原条目。

它也支持收藏、图片预览、OCR 和图片语义搜索。

## 为什么做

大多数剪贴板工具优化的是“找回原始内容”。Deep Index Board 更关注找回之后的那一步：内容差不多对了，但粘贴前还需要微调。

## 亮点

- 可编辑文本预览：修改后可另存为新条目，也可覆盖当前条目
- 历史列表上方的收藏区，支持收起、调整高度和一键取消全部收藏
- 图片预览、OCR 和图片语义搜索
- 单条删除与清空未收藏历史记录

## 已实现功能

### 剪贴板历史

- 自动记录文本、图片、文件和文件夹剪贴板内容
- 快捷键唤醒剪贴板面板，点击历史项即可粘贴
- 支持单条删除历史记录
- 支持清空未收藏的剪贴板记录，并删除应用内保存的图片文件

### 收藏区

- 支持将历史条目加入收藏
- 收藏区固定显示在历史列表上方，默认收起
- 收藏区可手动展开、收起，并支持拖动分界线调整高度
- 支持一键取消全部收藏

### 搜索与预览

- 支持关键词搜索历史内容
- 支持语义搜索，用文字描述查找历史图片
- 图片内容自动保存，并支持预览
- 图片和图片文件支持 OCR 文本提取
- 文本文件可显示内容快照

### 文本编辑

- 文本条目可在右侧预览区显式进入编辑状态
- 编辑后的内容需要手动选择“另存新条目”或“覆盖保存”
- 编辑草稿不会自动污染原始历史记录

### 应用体验

- 支持调整窗口大小、拖动窗口位置
- 支持开机自启动开关
- 底部显示当前内存占用

## 使用方式

启动后应用会在后台监听剪贴板。

- Windows：按 `Alt + V` 打开面板
- macOS：按 `Control + V` 打开面板

在面板中可以搜索、预览、收藏、编辑、删除和选择历史内容。顶部“清空”会清空未收藏的历史记录，收藏区中的内容会保留。

## macOS 未签名应用说明

当前项目会直接分发 `.app` 文件，不是经过签名和公证的安装包。只在你信任来源的情况下运行。

如果 macOS 提示应用来自身份不明的开发者，优先尝试：

1. 将 `deep-index-board.app` 移到 `/Applications`。
2. 右键点击应用，选择“打开”。
3. 如果 `系统设置` -> `隐私与安全性` 中出现“仍要打开”选项，点击它。

如果 macOS 仍提示应用已损坏或无法打开，可以移除隔离属性：

```bash
xattr -dr com.apple.quarantine /Applications/deep-index-board.app
```

如果应用放在其他位置，把路径换成实际的 `.app` 路径：

```bash
xattr -dr com.apple.quarantine "/path/to/deep-index-board.app"
```

应用还需要“辅助功能”权限来完成自动粘贴和焦点相关行为。请在 `系统设置` -> `隐私与安全性` -> `辅助功能` 中允许本应用。如果授权后仍无法粘贴，可以先从辅助功能列表中移除应用，再重新添加并重启应用。
