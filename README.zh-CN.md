# Deep Index Board

[English](README.md) | 简体中文

Deep Index Board 是一个面向日常使用的剪贴板历史工具。它会在后台记录你复制过的内容，并提供快速唤醒、预览、搜索、收藏、编辑和再次粘贴能力。

## 已实现功能

### 剪贴板历史

- 自动记录文本、图片、文件和文件夹剪贴板内容
- 快捷键唤醒剪贴板面板，点击历史项即可粘贴
- 支持单条删除历史记录
- 支持清空未收藏的剪贴板记录，并删除应用内保存的图片文件

### 收藏区

- 支持将历史条目加入收藏
- 收藏区固定显示在历史列表上方，默认收起
- 收藏区可手动展开、收起，并支持拖动分界线调整高度
- 支持一键取消全部收藏

### 搜索与预览

- 支持关键词搜索历史内容
- 支持语义搜索，用文字描述查找历史图片
- 图片内容自动保存，并支持预览
- 图片和图片文件支持 OCR 文本提取
- 文本文件可显示内容快照

### 文本编辑

- 文本条目可在右侧预览区显式进入编辑状态
- 编辑后的内容需要手动选择“另存新条目”或“覆盖保存”
- 编辑草稿不会自动污染原始历史记录

### 应用体验

- 支持调整窗口大小、拖动窗口位置
- 支持开机自启动开关
- 底部显示当前内存占用

## 使用方式

启动后应用会在后台监听剪贴板。

- Windows：按 `Alt + V` 打开面板
- macOS：按 `Control + V` 打开面板

在面板中可以搜索、预览、收藏、编辑、删除和选择历史内容。顶部“清空”会清空未收藏的历史记录，收藏区中的内容会保留。
